p

O "p" é uma tag de parágrafo, feita para grandes escritas como esse texto que você está lendo agora. Ele é uma tag padrão, então possui espaçamentos e convenções que o HTML provê automaticamente.

div

O "div" é uma tag de conteúdo como essa que você está lendo agora, porém não é ideal para escrever grandes escritas pois não possui as organizações que o "p" possui

a

O "a" é uma tag de link clicável, que é adicionada dentro de um texto, como por exemplo: Clique Aqui.

br

O "br" serve para pular linhas de forma simples, por exemplo:





strong

O "strong" é feito para deixar qualquer texto em negrito, como por exemplo dentro desse texto de "p" aqui

ul

O "ul" é feito para criar listas de forma simples:

  • Item1
  • Item2
  • Item3

ol

O "ol" é feito para criar listas enumeradas de forma simples:

  1. Item1
  2. Item2
  3. Item3

table

Dessa lista, o "table" é o mais complexo, ele cria tabela de forma organizada e simples:
